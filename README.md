# challenge_ForIT_2025

# 📝 Task  App

TaskApp es una aplicación web diseñada para ayudarte a organizar y gestionar tus tareas de manera simple, rápida y eficiente. Su objetivo principal es facilitar el seguimiento de actividades, permitiendo a los usuarios aumentar su productividad y mantener el control de sus objetivos diarios o semanales.

---


## ✅ Funcionalidades


- Crear, editar y eliminar tareas

- Marcar tareas como completadas o pendientes

- Filtro por estado: todas, pendientes, completadas

- Notificaciones y Toastify

## 🚀 Tecnologías utilizadas en el Frontend

- **React:** Librería de JavaScript para construir interfaces de usuario.
- **Material UI (MUI):** Framework de componentes de interfaz de usuario para React, basado en Material Design
- **React Router DOM:** Librería para manejar la navegación en aplicaciones React.
- **React Toastify:** Librería para mostrar notificaciones tipo "toast". React Toastify 
- **Styled Components:** Librería para escribir estilos en componentes de React.
---

## 🚀 Tecnologías utilizadas en el Backend

- **Express:** Framework web para Node.js, utilizado para construir el servidor backend. 
- **Day.js:** Librería para manipulación de fechas, ligera y fácil de usar.
- **dotenv:** Librería para cargar variables de entorno desde un archivo .env
- **CORS:** Middleware para habilitar solicitudes cruzadas entre dominios (Cross-Origin Resource Sharing).
- **Nodemon:** Herramienta de desarrollo que reinicia automáticamente el servidor cuando se detectan cambios en los archivos.
---

## 🛠️ Requisitos previos

- Node.js (v16 o superior)
- npm o yarn
- Git

---

## 📦 Instalación

### 1. Clonar el repositorio

 Primero, abre tu terminal o consola de comandos y clona el repositorio utilizando el siguiente comando:


 git clone **https://github.com/JJMDT/Challenge_ForIT2025**

Una vez clonado el repositorio, sigue estos pasos para instalar las dependencias tanto del frontend como del backend.

cd challenge_ForIT2025/backend

Instala las dependencias del backend:


- npm install

Accede al directorio del frontend:

- cd ../frontend
Instala las dependencias del frontend:

- npm install

Ejecutar el proyecto
Regresa al directorio del backend:


- cd backend
Ejecuta el servidor backend con el siguiente comando (en modo de desarrollo):


- npm run dev
Regresa al directorio del frontend:


- cd ../frontend
Ejecuta el servidor backend con el siguiente comando (en modo de desarrollo):

- npm run dev

Una vez que ambos servidores estén en ejecución, puedes acceder a la aplicación en el navegador utilizando la siguiente URL:

📍 **http://localhost:3000**
